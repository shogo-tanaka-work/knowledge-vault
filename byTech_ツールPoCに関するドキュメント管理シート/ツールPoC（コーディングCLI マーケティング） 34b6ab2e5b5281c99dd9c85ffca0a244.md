# ツールPoC（コーディングCLI*マーケティング）

ステータス: 保留

**カテゴリ:** ツールPoC（CLIマーケティング）

**期間:** 2025/07/05 - 2025/07/11

**プロジェクトステータス:** 保留（手法の再検討）

### 1. 背景と目的

Claude CodeやGemini CLIを用い、ローカルのMarkdownファイルを読み込ませることで、経営分析やマーケティング施策立案の効率化を検証する。

### 2. 取り組み内容

- **実施施策:** CLIツール（コマンドライン）を用いて、マーケティング施策案の生成および.mdファイルへの出力を実行。
- **使用ツール・技術:** Claude Code, Gemini CLI
- **主要な意思決定:** 従量課金のClaudeと無料枠のあるGeminiの比較検証。

### 3. 進捗と成果

- **達成できたこと:** CLI経由でのファイル読み込み・生成フローの確認。
- **定性的な成果:** * Geminiの方が細かな出力が得られた。
    - Claude Codeは従量課金、Gemini CLIは無料というコスト差を確認。
    - コンテキスト保持やMCP連携の可能性など、拡張性の高さは確認。

### 4. 学びとナレッジ

- **うまくいったこと（Good）:** ローカル情報をRAG（外部知識参照）のように活用できる手応えを得た。
- **改善ポイント（Improve）:** 「なぜCLIである必要があるのか？」という問いに対し、非エンジニアにはハードルが高いという結論に至った。

### 5. 課題と対応

- **発生した課題:** CLI操作への不慣れさ。
- **対応方法:** 一般的なマーケティング用途であれば、通常のチャットUI（推論モデル）で十分であると整理。

### 6. 今後の展開

- **横展開の可能性:** エンジニアがコードベースを理解した上でマーケティング資料を生成する際には有効。
