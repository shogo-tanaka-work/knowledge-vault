# Webflow / Stitch / Figma 連携によるデザインインポート検証

**期間:** 2025/10/02

**プロジェクトステータス:** 一部保留（不具合確認）

### 1. 背景と目的

- **背景:** Figmaで作成したデザインをWebflowへインポートし、コーディングなしでのサイト公開フローを検証。

### 4. 学びとナレッジ

- **発生した課題:** プラグインを用いたインポート時、横並び要素が縦並びに変換される不具合が発生。
- **対応方法:** Webflow側の構造定義（Flexbox等）とFigmaのAuto Layoutの整合性を再調整する必要があるが、現状は手動修正の工数が大きいため優先度を下げて保留とした。
