# Gemini File Search API による RAG（検索拡張生成）環境の構築

ステータス: 完了

**カテゴリ:** ツールPoC（AIエージェント・RAG）

**期間:** 2025/11/01 - 2025/11/07

**プロジェクトステータス:** 完了（小規模アプリ向け実装パターン確立）

### 1. 背景と目的

- **背景:** 大量の独自ドキュメントを参照するRAG環境を、Vertex AI等の複雑な基盤なしにAPIのみで構築できるか検証。

### 2. 取り組み内容

- **実施施策:** Gemini APIのFile Searchツールを用い、ファイルのアップロード→ストア構築→API問い合わせのフローを実装。
- **使用ツール・技術:** Gemini API (File Search Tool), JavaScript (Node.js)

### 3. 進捗と成果

- **達成できたこと:** 約50行程度のコードで、独自ソースに基づく回答生成（RAG）を実装完了。
- **定量的な成果:** 開発スピードにおいて、従来のVector DB構築を伴うRAGより圧倒的に早いことを確認。

### 4. 学びとナレッジ

- **技術的な発見:** 非エンジニアには難易度が高いが、エンジニアにとってはモック環境や小規模アプリへのRAG実装として極めてコスパが高い。
