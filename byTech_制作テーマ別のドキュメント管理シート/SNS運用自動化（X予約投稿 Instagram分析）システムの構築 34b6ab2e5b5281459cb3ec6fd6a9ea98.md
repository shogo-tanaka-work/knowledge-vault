# SNS運用自動化（X予約投稿 / Instagram分析）システムの構築

カテゴリ: マーケティング
ステータス: 進行中

**カテゴリ:** ツールPoC（マーケティング自動化・SNS）

**期間:** 2025/09/23 - 2025/09/24

**プロジェクトステータス:** X予約投稿のみ完了（基盤構築完了）

### 1. 背景と目的

- **開始理由:** SNS（X, Instagram, TikTok）の投稿漏れ防止、およびアナリティクス情報の収集・分析を自動化し、戦略立案に集中できる環境を作るため。
- **解決したい課題:** 投稿タイミングの逸失、手動でのインサイト収集にかかる手間の解消。

### 2. 取り組み内容

- **実施施策:** 1. GASとX API v2を用いたスプレッドシート連動型予約投稿システム。
    1. Instagram Graph APIを用いたビジネスアカウントのインサイト自動取得。
    2. 取得した分析結果のDiscordへの自動通知。
- **使用ツール・技術:** GAS, X API v2, Instagram Graph API, Facebook Developer Portal

### 3. 進捗と成果

- **定量的な成果:** 15分程度の設定でXの予約投稿環境が構築完了。Instagramのインサイト取得までを2時間で実装。
- **定性的な成果:** 投稿内容の蓄積をDB（スプレッドシート）化することで、過去のバズ傾向との照らし合わせが容易になった。

### 4. 学びとナレッジ

- **技術的な発見:** Instagram Graph API（Meta）は設定手順が非常に煩雑（Facebookページ連携等）だが、一度繋がればcurlやGASで容易に操作可能。
- **セキュリティ:** アクセストークンの漏洩リスクを考慮し、2ヶ月ごとの交換（または管理の厳格化）が必須。

### 5. 課題と対応

- **発生した課題:** Xの無料枠（月1,500ツイート）内での画像・動画投稿の制限、および投稿内容そのものを考える手間の残存。
- **対応方法:** LLM APIを連携させ、過去の成果から「反応の良い投稿内容」をAIが下書きするフローを検討。

### 6. 今後の展開

- **次のアクション:** TikTok APIの本格検証、および全SNSの指標を横断的に閲覧できる「SNSダッシュボード」の構築。
