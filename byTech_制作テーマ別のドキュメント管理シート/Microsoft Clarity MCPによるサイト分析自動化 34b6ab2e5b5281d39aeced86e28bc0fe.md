# Microsoft Clarity MCPによるサイト分析自動化

カテゴリ: マーケティング
ステータス: 完了

**カテゴリ:** ツールPoC（マーケティング・分析）

**期間:** 2025/08/02 - 2025/08/08

**プロジェクトステータス:** 完了（実用化推奨）

### 1. 背景と目的

- **開始理由:** バイテック公式HPのエンゲージメント情報を、ダッシュボードにアクセスせずMCP経由で取得するため。
- **期待される効果:** クラリティへのログイン工数を削減し、チャットUI上で迅速なサイト改善の意思決定を行う。

### 2. 取り組み内容

- **実施施策:** Clarity MCPサーバーを構築し、Claude Desktopからサイト訪問者の挙動データを取得。
- **使用ツール・技術:** Claude Desktop, Clarity MCP server
- **主要な意思決定:** 先のGA4検証と同様の操作感を目指し、主要なヒートマップデータやセッション情報の取得を検証。

### 3. 進捗と成果

- **達成できたこと:** チャット経由でのエンゲージメント情報の取得。
- **定性的な成果:** GA4 MCPと同等の利便性を確認。ただし、GA4同様にGoogle広告との直接連携情報は取得できないという制約を特定。

### 4. 学びとナレッジ

- **うまくいったこと（Good）:** 特定のページURLを指定するだけで、そのページのユーザー行動（デッドクリック等）を要約できる。
- **技術的な発見:** Microsoft公式のMCPサーバーを活用することで、安定したデータ連携が可能。

### 5. 課題と対応

- **発生した課題:** 広告経由の流入分析の限界。
- **対応方法:** 広告分析については、別途Google Ads用のソリューションが必要であると定義。

### 6. 今後の展開

- **横展開の可能性:** 自社サイト運営を行っているToBクライアントへ、GA4 MCPとセットでの導入提案。
