# 最新AI情報取得および教材更新判断の自動化検証

ステータス: 進行中

**カテゴリ:** ツールPoC（教育DX・ナレッジ管理）

**期間:** 2025/11/08 - 2025/11/21

**プロジェクトステータス:** ロジック修正フェーズ

### 1. 背景と目的

- **背景:** 主要AIツールのアップデートを目視で追う限界を解消し、教材やコミュニティへの反映が必要な情報を自動抽出する。

### 2. 取り組み内容

- **検証内容:** LangGraphを用いた、各ツールのリリースノート（RSS等）の自動監視とDiscord通知。

### 4. 学びとナレッジ

- **発生した課題:** Microsoft Copilot等のRSSでは「詳細リンク」の情報しか取れず、中身の解析が不十分。
- **対応方法:** 詳細リンク先までAIが巡回して内容をスクレイピング・要約するステップをロジックに追加する。
