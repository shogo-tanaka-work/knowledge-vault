# 最新AI情報取得および教材更新判断の自動化検証

ステータス: 進行中

**カテゴリ:** ツールPoC（教育DX・ナレッジ管理）

**期間:** 2025/12/12 - 2025/12/19

**プロジェクトステータス:** ロジック改修中

### 1. 背景と目的

- **背景:** 教材の鮮度を保つため、主要ツールのアップデート情報を自動監視し、Discordへ通知する。

### 4. 学びとナレッジ

- **発生した課題:** Microsoft Copilot等のリリースノートにおいて、RSSで取得できるのは「サイトへのリンク」のみであり、具体的な更新内容が不足している。
- **対応方法:** 詳細リンク先をAIにクローリングさせ、変更点を要約するステップの追加を検討。
