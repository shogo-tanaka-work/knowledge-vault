# Google Custom Search APIによる競合・オーガニック動向の自動取得

カテゴリ: マーケティング
ステータス: 完了

**カテゴリ:** ツールPoC（マーケティング・SEO分析）

**期間:** 2025/09/27

**プロジェクトステータス:** 完了（マニュアル化可能）

### 1. 背景と目的

- **開始理由:** 特定キーワードの検索結果（オーガニック）を自動収集し、競合の変動やトレンドを定点観測するため。
- **期待される効果:** 手動検索の手間をなくし、戦略立案の素材となる生データを自動で蓄積する。

### 2. 取り組み内容

- **実施施策:** Google Cloud Console（GCC）でCustom Search APIを有効化し、GASを用いて検索結果をDiscordへ通知する仕組みを構築。
- **使用ツール・技術:** Google Custom Search API, GAS, Discord

### 3. 進捗と成果

- **達成できたこと:** 特定の検索エンジンIDとAPIキーを用いた、キーワード別ランキングデータの自動取得。
- **定性的な成果:** 操作がシンプルであるため、検索条件の追加作業を容易にマニュアル化できる状態となった。

### 6. 今後の展開

- **次のアクション:** 取得したタイトル・スニペット情報をLLMに渡し、「競合の訴求ポイントの変化」を週次でレポート化する処理を追加する。
