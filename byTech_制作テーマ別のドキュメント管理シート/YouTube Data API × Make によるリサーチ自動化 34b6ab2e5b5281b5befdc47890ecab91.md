# YouTube Data API × Make によるリサーチ自動化

カテゴリ: マーケティング
ステータス: 完了

**カテゴリ:** ツールPoC（業務自動化・SNSリサーチ）

**期間:** 2025/09/16 - 2025/09/17

**プロジェクトステータス:** 完了（汎用ロジック確立）

### 1. 背景と目的

- **開始理由:** YouTube上のバズ動画リサーチや競合分析を自動化したいという受講生・SNS担当者のニーズに応えるため。
- **解決したい課題:** 手動での動画収集・数値チェック（再生数・投稿日等）にかかる膨大な工数の削減。

### 2. 取り組み内容

- **実施施策:** Makeの標準モジュールでは詳細検索に限界があったため、YouTube Data APIを直接叩く「API Call」を用いたカスタムロジックを構築。
- **使用ツール・技術:** Make, YouTube Data API v3

### 3. 進捗と成果

- **定量的な成果:** 動画収集にかかる作業時間をほぼ「ゼロ」に短縮。条件に合致する動画リストをスプレッドシート等へ即座に蓄積可能に。
- **定性的な成果:** 検索パラメータの調整（投稿期間、言語、並び順）により、精度の高いリサーチ環境を実現。

### 4. 学びとナレッジ

- **技術的な発見:** Makeの標準機能に頼らずAPI Callを利用することで、クォータ（利用制限）を節約しつつ高度なフィルタリングが可能。

### 5. 課題と対応

- **発生した課題:** APIの仕様理解が必要なため、非エンジニアへの展開にはマニュアル化が必須。
- **対応方法:** 設定値をパラメータ化し、ユーザーがキーワードや期間を書き換えるだけで済むテンプレートを作成。

### 6. 今後の展開

- **次のアクション:** 同様のロジックをGASへ移植し、実行コストの完全無料化とさらなる柔軟性を追求する。
