# Threads連投｜J-Net21補助金自動化やってみた

## 投稿1（気づきベース）

補助金リサーチって毎月地味に時間溶ける作業だったんだけど、
Claude Code にやらせたら5分で終わった。

J-Net21（中小機構のサイト）を Browser Use CLI で開いて、全件取って、募集期間で絞って、Excelにまで落とすところまで全部おまかせ。

「あ、これでよかったんだ」感がすごい。

---

## 投稿2（試した話）

やり方、最初はチェックボックスをクリックする方針で考えてたんだけど、
これが全然うまくいかなくて。フォームsubmit押すと別のページに飛ばされる謎仕様。

途中で「URLパラメータ直接組むほうが早くね？」と気づいて方針転換。
これが正解だった。`field[]` `type[]` `period=` で全部制御できた。

---

## 投稿3（詰まったところ）

ドキュメントに載ってる `browser-use extract` が実装されてなかったの、地味にショックだった。

代わりに `eval` で JavaScript 流す形で全部書き直し。
結果的にこっちのほうが柔軟だったから結果オーライなんだけど、最初の30分は無駄にした。

---

## 投稿4（次やりたいこと）

これ、補助金以外にも普通に使えるなと思ってて。

・公募情報の定期ウォッチ
・競合の新着情報チェック
・求人情報のフィルタ収集

「定期的に同じサイト見に行って条件で絞る」みたいな作業、世の中にめちゃくちゃあるはず。
順番に潰していきたい。
