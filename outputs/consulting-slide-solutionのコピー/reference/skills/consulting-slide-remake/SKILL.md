# Consulting Slide Remake — 戦略コンサル風スライド再現プロンプト集

このスキルは、参考にしたい1枚のスライド画像から「同等の構造・配色・コンサル品質を再現するためのプロンプト」を生成する。プロンプトの実行（実際の画像生成 / PPTX生成）はこのスキルでは行わず、ユーザーが好みのモデル・ツールに投げる前提。

## 想定ユースケース

- 大手コンサル（アビーム、マッキンゼー、BCG、NRI、三菱総研、Deloitte 等）が出すレベルのスライドを、自分のテーマで再現したい
- 良い参考スライドの画像はあるが、ゼロから言語化するのが大変
- 「画像→画像」だと崩れるので、いったん「構造を解析した精緻なプロンプト」を経由したい
- 画像で完成したものを、後で PowerPoint で編集できる .pptx として再構築したい

## モード切替

ユーザーの依頼内容で2モードを使い分ける。

### Mode A — 画像生成AI向け「再現プロンプト」生成
- 発動キーワード: 「コンサル風スライドを再現」「画像から再現プロンプト」「アビーム風で作って」「同じレイアウトで自分のテーマで作りたい」
- 出力先: `prompts/A-remake-prompt.md` の手順に従い、戦略コンサル風スライド生成プロンプトを **本文のみ** で出力
- 出力先AI想定: ChatGPT (GPT-Image), Claude (Design tool), Gemini (Image), nano banana 等

### Mode B — 編集可能PPTX再構築プロンプト生成
- 発動キーワード: 「これをパワポにして」「画像を編集可能な pptx に」「PDF を直接編集できる pptx に」「ズレなく PPTX 化」
- 出力先: `prompts/B-pptx-conversion.md` の手順に従い、画像/PDF を pptx として再構築するための工程指示を出力
- 出力先AI想定: Claude Code（python-pptx 経由）/ Cursor / Codex 等のコード生成エージェント

## 実行手順

1. **入力確認**: ユーザーから1枚の参考スライド画像（または PDF）が提示されているかを確認する
2. **モード判定**: 依頼文から Mode A / Mode B を決定。曖昧な場合のみユーザーに確認
3. **プロンプト本文を読み込み**:
   - Mode A → `prompts/A-remake-prompt.md`
   - Mode B → `prompts/B-pptx-conversion.md`
4. **画像解析**: マルチモーダル能力で画像を読み込み、プロンプト内の指示（構造解析・bbox抽出・配色取得等）に従う
5. **出力**: プロンプト内の `<output_format>` または `<critical_rules>` を厳守し、前置き・後置き・コードフェンスなしで本文のみ返す

## 連携先スキル

- `slide-design-patterns` — テーマ別のスライド構成パターン辞書。Mode A 出力の構成判定で参照する
- `majin-slide-pptx` / `navy-blue-pptx` / `navy-blue-pptx-json` — Mode B 出力を実装する際の python-pptx 生成スキル
- `outputs/html-slides/` — 当スキルと並行で進行中の HTML テンプレート集。コンサル風配色・タイポの参考実装

## 出典

X 投稿（2026年5月）にて共有された「アビームの業務フロー資料を AI で再現する」用プロンプト2本。原文を一次資料としてプロジェクト内に保管した上で、スキル化している。
