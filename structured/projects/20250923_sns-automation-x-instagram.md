## 1. 背景と目的
* 開始理由: SNS（X, Instagram, TikTok）の投稿漏れ防止、およびアナリティクス情報の収集・分析を自動化し、戦略立案に集中できる環境を作るため。
* 解決したい課題: 投稿タイミングの逸失、手動でのインサイト収集にかかる手間の解消。

---

## 2. 取り組み内容

### 実施した施策・活動
* GASとX API v2を用いたスプレッドシート連動型予約投稿システム。
* Instagram Graph APIを用いたビジネスアカウントのインサイト自動取得。
* 取得した分析結果のDiscordへの自動通知。

### 使用したツール・技術
* GAS
* X API v2
* Instagram Graph API
* Facebook Developer Portal

### 技術的な発見・Tips
* Instagram Graph API（Meta）は設定手順が非常に煩雑（Facebookページ連携等）だが、一度繋がればcurlやGASで容易に操作可能。
