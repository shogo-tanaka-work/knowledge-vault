## はじめに

「ChatGPTで聞けばコードは書ける。でも、なぜか自分のアプリは完成しない」  
そんな経験はありませんか？

私はこれまで **生成AI講師として40名以上の方を個別指導** してきました。  
そこで見えてきたのは、独学でつまずく原因の多くが「スキル不足」ではなく **"伴走者の不在"** だということです。

このプランでは、**Claude Code と Codex** という、2026年現在の AI 駆動開発の2強を使って、  
あなたの「作りたい」を **URLで共有できる"動くアプリ"** にするまでを、4週間〜伴走します。

---

## こんな方に向いています

- ChatGPT・Claude などで何度かコードを書いてもらったが、完成までたどり着けなかった方
- 独学で3ヶ月以上頑張ったが、途中で詰まってしまった方
- 「作りたいもの」は明確にあるけれど、技術選定で止まってしまっている方
- AI駆動開発（Claude Code / Codex）を体系的に身につけたい方
- 副業・個人開発の第一歩として、まず1つアプリをリリースしたい方

> 独学で何度も詰まった経験を否定しません。むしろ、そこを起点に伴走します。

**Q. プログラミング未経験でも大丈夫ですか？**  
A. 大丈夫です。ChatGPTなどのAIに触れたことがあれば十分スタートできます。Claude Code・Codexの導入から一緒に進めます。

**Q. 働きながらでも続けられますか？**  
A. 週3〜5時間の確保で進められる設計です。チャット相談は24時間以内返信なので、平日夜・週末で十分回せます。
