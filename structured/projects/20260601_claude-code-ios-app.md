## 1. 背景と目的
* 筋トレの記録を分析と習慣化につなげる個人開発アプリ（Workout Habit）を、自分用 iOS アプリとして形にしたい
* AIコーディング（Claude Code）だけで、企画・設計・実装・ビルド・実機導入までどこまで通せるかを検証する
* 「PoC を外注する前に、まず動くものを自分の手元で作って確かめられるか」という問いの一つの答えにする

---

## 2. 取り組み内容

### 実施した施策・活動
* Expo（React Native）で記録アプリの MVP を実装（ワークアウト記録・種目/セットのCRUD・履歴サマリ・休憩タイマー）
* Xcode 導入後、iOS development build をシミュレータで起動
* 日本語パス起因のビルド失敗を、プロジェクトのフォルダ名英語化（ASCII化）で解消
* 仕様（MVPスコープ/要件/画面設計）と実装の突き合わせレビュー
* フィードバックを受けて記録UXを改修（一時保存・履歴の再編集・休憩時間のホイール選択）
* 無料 Apple ID でのローカル署名により、実機（自分のiPhone）へ Release 版を単体動作でインストール

### 使用したツール・技術
* Claude Code（設計・実装・ビルド実行・デバッグ・記録）
* Expo SDK 56 / React Native 0.85 / TypeScript
* expo-sqlite（ローカル保存）、expo-audio（タイマー終了音）、@react-native-picker/picker（ホイール選択）
* Xcode 26.5 / CocoaPods 1.16.2 / Watchman / Node 22

### 主要な意思決定とその理由
* CocoaPods はメンテナンスモード入りだが、Expo SDK 56 の `expo run:ios` が前提とするため採用（SPM 移行はツールチェーン側の将来課題）
* repo を移設せず、フォルダ名を英語化してビルドパスを ASCII 化（マルチバイトパス問題の根治）
* 実機常用のため、Metro 依存の dev build ではなく JS 内蔵の Release ビルドを採用

## 4. 学びとナレッジ

### うまくいったこと（Good）
* 記録ループの実装は素直に通った。SQLite 永続化・都度保存・履歴集計まで一気通貫
* エラーログから原因（prebuilt tarball の URI 化失敗）を特定し、フォルダ英語化で根治できた
* 署名が一度通れば、以降はターミナルから再ビルド・再インストールできる

### うまくいかなかったこと（Bad）
* 日本語フォルダ名で iOS ネイティブビルドが失敗。Apple ツールチェーンはマルチバイトパスに弱い
* `npm run ios:dev --device` は `--` を入れないと引数が expo に渡らず、シミュレータに入ってしまう
* ターミナルからの codesign が `errSecInternalComponent` で失敗（キーチェーンの鍵アクセス）

### 改善ポイント（Improve）
* モバイル系プロジェクトは最初から ASCII パスで作る
* 実機の初回ビルドは Xcode の Run（常に許可）で署名 ACL を解決してから、ターミナルに戻る

### 技術的な発見・Tips
* RN 0.85 の prebuilt artifacts はローカルパスを `file://` URI 化するため、日本語が混ざると `bad component` で落ちる
* iOS 16+ は実機開発に端末側「デベロッパモード」が必須
* dev build は Mac の Metro が必要。ジムなど単体利用には Release ビルド（JS 内蔵）が必要
* 無料 Apple ID はアプリが7日で期限切れ。切れたら同コマンドで再インストール

---

## 5. 課題と対応

### 発生した課題
* 日本語パスでのビルド失敗 / Developer Mode 未有効 / キーチェーン署名拒否 / dev build の Metro 依存

### 対応方法
* フォルダ名を英語化（`10_プロジェクト`→`10_projects`、`筋トレ習慣化アプリ`→`workout-habit` ほか）してパスを ASCII 化
* iPhone 設定でデベロッパモードを有効化して再起動
* Xcode から一度 Run して codesign を「常に許可」、以降はターミナル署名も通過
* 単体動作のため Release ビルドを実機にインストール

### 未解決の課題
* Phase 1 完了条件のうち「前回実績の参照（入力画面）」と「種目別の推移（種目詳細＋簡易グラフ）」が未達
* 無料枠のため7日でアプリが失効する（常用には Apple Developer Program + EAS Build が必要）
