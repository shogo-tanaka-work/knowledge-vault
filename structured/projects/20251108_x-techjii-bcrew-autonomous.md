## 1. 背景と目的
* 開始理由: 最新AI情報の収集・加工・発信を完全に自動化し、24時間稼働の「AIインフルエンサー」として運用するため。
* 解決したい課題: 手動投稿の限界、および情報の鮮度とエンゲージメントの両立。

---

## 2. 取り組み内容

### 実施した施策・活動
* Node.js基盤への移行（Render公開）。
* テキスト＋画像付きツイートの自動化。
* 独自ロジックによるエンゲージメント計算（上位5件抽出）。

### 使用したツール・技術
* Node.js
* Render
* X API
* Gemini API（Imagen/Nano）

### 主要な意思決定とその理由
* n8nのみでは画像生成・投稿の自由度が低いため、Node.jsを基盤としたシステム構築へ舵を切った。
