## 3視点クリティカルレビュー（v2 新規）

省伍さんの提案を、以下3名の立場から見直しました。フィードバックは「より良くするための提案」で、否定ではありません。

### 👤 木村さん（ライター歴20年）視点

**良いところ**
- 「作りたいものがある。でも、一人では完成しない人へ」は本当に良いコピー。これは武器として残すべき。
- 数値・表での現状把握が綺麗で、診断書として読める。

**もう少し磨ける点**
1. **タイトル案がやや説明的**。MENTA人気プランの公式は「【超初心者OK】Claude Code 個別サポート」のように、**タグは短く、ベネフィットは数値で**まとめる傾向。
2. **キャッチコピーが説明文の中に埋もれている**。これは"見出し級"の言葉。**タイトル直下のH2に昇格**させて1行で見せた方が刺さる。
3. **「独学でつまずく根本原因」の文章は良いが長め**。「箇条書き3点 → 解像度ある共感の一文」の二段構えにすると離脱が減る。
4. **「作り切る」がやや抽象的**。本文では「リリースまで」「URLで共有できる状態まで」など具体語に言い換えると体感が上がる。

### 📊 Web歴20年マーケター視点

**良いところ**
- 離脱原因分析が定量で、優先度設計（最高/高/中）も妥当。
- CVR目標を明確に置いている点が良い。

**もう少し磨ける点**
1. **口コミ依頼の打診文テンプレがない**。「個別に依頼」だけだと実行率が落ちる。**コピペで送れる文面**を同梱すべき（後述）。
2. **価格未記載は問題だが、根拠付き設計が必要**。MENTA人気帯は「単発1,000〜5,000円 / 月額チャット3,000〜15,000円 / **スクール型30,000〜50,000円/月**」。**省伍さんの権威性（生成AI講師40名指導）ならスクール型に寄せて問題なし**。
3. **ファネル設計が弱い**。3プラン構造は良いが、**「お試し→本格への引き上げ動線」の文言**が必要。
4. **投稿テーマが機能説明寄り**。MENTA投稿はSEOよりプラットフォーム内回遊が主。**「検索キーワード」より「共感ストーリー」**でCTRが伸びる。
5. **権威性の見せ方が"40名指導"止まり**。人気プランは「**実績数値 + 現役属性 + 経験年数**」の三点セット。年数を必ず入れる。
6. **CVR目標が控えめ**。改善余地5〜10倍と自己診断しているのに目標2倍は弱い。**1ヶ月後 1.5%、3ヶ月後 3.0%** が妥当。

### 🎓 受講候補者（独学でバイブコーディング挫折／熱量高め／軽い気持ち層）視点

**ページを見たときに感じること**
1. **「Claude Code」「Codex」と書かれても、自分が触ったツールと一致するか不安**。「ChatGPT で書いてもらってたけど完成しない人」のように、**自分の経験で受け止められる言葉**が欲しい。
2. **「作り切る」が抽象的**。「**URLで共有できる/友達に見せられる/ストアに公開できる**」のような具体ゴールがあると安心。
3. **熱量高い層は「価格より、ついていけるか」が不安**。「未経験でも大丈夫？」「働きながら続く？」のFAQが冒頭に欲しい。
4. **軽い気持ちの層は1ヶ月契約が重い**。**初回30分無料相談**を最も目立つ場所に。「相談だけでもOK」の温度感。
5. **挫折者は"否定されない場所"を探している**。「独学で何度も詰まった経験を否定しません」の一言が刺さる。

## 改善提案（修正版・優先度順）

### 1. 既存受講者5名に口コミ依頼【最優先・工数30分】

口コミ0件が最大の離脱要因。5.0満点という強みを活かすために即実施。

**アクション**
- 既存受講者5名にコピペ可能な依頼文（後述「口コミ依頼文テンプレ」）を送信
- 目標: 2〜3件の口コミ獲得

---

### 2. 価格の明示化【最優先・工数10分】

**改善後の推奨構成（確定版）**

```
【プラン1】無料相談（30分）         ¥0
　→ 作りたいものと現状レベルを共有。Claude Code・Codexの選定と方向を整理

【プラン2】お試しセッション         ¥5,000（90分・1回）
　→ Claude Code または Codex で実際に手を動かす。プロンプト設計の基礎を体験

【プラン3】本格伴走プラン           ¥29,800/月
　→ チャット24時間以内返信＋週1回ビデオ通話60分。企画〜リリースまで4週間〜伴走
```

**根拠**: MENTA人気帯のスクール型は3〜5万円/月。生成AI講師40名指導という権威性なら ¥29,800 は通る。お試しは ¥5,000 で心理ハードルを下げる。

**引き上げ動線（追記推奨）**: お試し終了時に「ここまでで〇〇できました。本格プランなら△△まで進められます」と差分を見せる。

---

### 3. タイトル & H2キャッチコピー【最優先・工数10分】

**現状**: `【AI駆動開発】Claude CodeやCodexを使って、自分のアプリを作り切る`

**修正後（最終案）**

```
タイトル:
【独学挫折OK】Claude Code・Codexで"動くアプリ"まで4週間伴走

H2（タイトル直下に昇格）:
作りたいものがある。でも、一人では完成しない人へ。
```

**意図**
- AI駆動開発の2強（Claude Code / Codex）をタイトルに残し、Cursorなど他ツールは本文「対象者」欄で軽く触れる程度に
- 「4週間」で期間ベネフィットを数値化（MENTA人気プランの型）
- 【独学挫折OK】で対象者タグを明示
- 既存の良いコピーを見出し級に格上げして"視認性"を確保

---

- もしくは「独学の限界を超える / 4週間で動くアプリ」を入れたバナー

```
   平日夜・週末で十分回せます。
```
