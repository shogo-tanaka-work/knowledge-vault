## 改善提案（優先度順）

### 1. 既存受講者5名に口コミ依頼【最優先・工数30分】

口コミ0件が最大の離脱要因。5.0満点という強みを活かすためにも即実施。

**アクション**:
- 既存受講者5名に個別メッセージで口コミ依頼
- 「率直な感想で構いません」という一言を添える
- 目標: 2〜3件の口コミ獲得

---

### 2. 価格の明示化【最優先・工数10分】

**現状**: 3プランの価格が未記載  
**問題**: 申し込みハードルが上がる、不信感につながる

**改善内容**:
- 各プランに具体的な金額を記載
- 「無料相談あり」を価格の前に見せて心理的ハードルを下げる

**推奨構成**:
```
【プラン1】無料相談 — ¥0
  → 作りたいものと技術レベルを共有。ツール選定と開発方向を整理

【プラン2】お試し体験 — ¥〇〇〇〇
  → AI駆動開発の進め方を体験。プロンプト設計の基礎を確認

【プラン3】本格実践 — ¥〇〇〇〇/月
  → 週1回のマンツーマン面談+チャットサポートで企画〜運用まで伴走
```

---

### 3. MENTAへの投稿を開始【高優先・工数60分×2本】

フォロワー0・投稿0はプラットフォーム内での信頼性ゼロを意味する。

**推奨テーマ**:
- 「Claude Codeを使ってWebアプリを30分で作った話」
- 「独学でアプリが完成しない本当の理由」

**目標**: 月2〜4本のペースで投稿 → フォロワー獲得 → プラン閲覧増

---

### 4. メイン画像の設定【高優先・工数30分】

**現状**: 画像なし → 一覧表示でクリックされにくい

**改善内容**:
- Claude Code の実際の動作画面スクリーンショット
- または文字入りバナー（「AIで作り切る」「独学の限界を超える」等）

**参考**: 競合はプランイメージ画像を設定しており、一覧での視認性・クリック率に直結

---

### 5. タイトルのターゲット明示化【高優先・工数10分】

**現状**: `【AI駆動開発】Claude CodeやCodexを使って、自分のアプリを作り切る`  
**問題**: ターゲット（誰向けか）が不明

**改善案（いずれか選択）**:

```
案A: 【独学挫折者向け】Claude Code/Codexで、アイデアを動くアプリに作り切る

案B: 【AI駆動開発】コードは書けるのに完成しない人へ——Claude Codeで作り切る

案C: 【完成まで伴走】AI駆動開発で、独学では届かなかったアプリを作り切る
```

> キャッチコピー「作りたいものがある。でも、一人では完成しない人へ」はタイトルに昇格させることも検討

---

### 6. 「できること」セクションの3カテゴリ化【中優先・工数20分】

**競合の構造（参考）**:
1. Webアプリケーション開発（フレームワーク）
2. サーバー構築（Linux/AWS）
3. チーム開発（Git/GitHub）

**省伍さん向け改善案**:
1. **AI駆動開発のセットアップ** — Claude Code / Codex の導入・使いこなし
2. **アプリ設計〜完成まで** — 企画・構成・デプロイ・運用
3. **LLM活用の実践知識** — コスト管理・エラーハンドリング・プロンプト設計

---

### 7. 返信速度・サポート体制の明示【中優先・工数10分】

**現状**: 返信速度の記載なし（競合は「24時間以内」を明示）

**追記内容**:
```
・チャット返信：基本24時間以内（平日）
・週1回のビデオ通話でリアルタイム相談可
```

---

### 8. 社会的証明の前面化【中優先・工数15分】

**現状**: 実績が下部のプロフィール欄に埋もれている

**改善内容**:
- 「生成AI講師として40名以上指導」をプラン説明の冒頭に記載
- 「現時点で全レビュー満点（5.0）」を強調表示

---

### 9. CTA動線の強化【中優先・工数10分】

**現状**: CTAが「メッセージで相談」と「事前に質問する（無料）」のみ

**改善内容**:
- プラン説明文の最後に誘導文を追加
  ```
  まず何を作りたいか話すだけでOKです。→ 無料で相談してみる
  ```
- 無料プランを先頭に持ってきて、最初のアクションを明確化

## 推奨ページ構成（リニューアル版）

```
1. タイトル（ターゲット明示）
2. メイン画像（Claude Code画面 or バナー）
3. キャッチコピー「作りたいものがある。でも、一人では完成しない人へ」

5. できること（3カテゴリ）
6. プラン・価格一覧（無料相談 / お試し / 本格実践）
7. メンタープロフィール（実績・資格・指導歴40名以上）
8. レビュー・口コミ（口コミ獲得後に機能する）
9. よくある質問（あれば）
10. 無料相談CTA「まず話すだけでOK →」
```
