## 1. 背景と目的
* 開始理由: GA4やClarity、Google Adsの散らばったデータを自動集計し、人間が分析する前にAIが「改善の兆し」や「異常値」を提案する仕組みを完成させるため。
* 期待される効果: 分析業務のルーチン化解除と、広告ROIの最大化。

---

## 2. 取り組み内容

### 実施した施策・活動
* リフレッシュトークンを用いたOAuth認証の完全自動化。
* Python実行環境（Pydantic関連）の不具合解消。
* ローカルMCP環境の共有および、リモート環境への移行調査。

### 使用したツール・技術
* GA4 API
* Clarity API
* Google Ads API
* Python (MCP)
* GAS
