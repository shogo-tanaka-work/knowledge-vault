## 1. 背景と目的
* 開始理由: YouTube上のバズ動画リサーチや競合分析を自動化したいという受講生・SNS担当者のニーズに応えるため。
* 解決したい課題: 手動での動画収集・数値チェック（再生数・投稿日等）にかかる膨大な工数の削減。

---

## 2. 取り組み内容

### 実施した施策・活動
* Makeの標準モジュールでは詳細検索に限界があったため、YouTube Data APIを直接叩く「API Call」を用いたカスタムロジックを構築。

### 使用したツール・技術
* Make
* YouTube Data API v3
