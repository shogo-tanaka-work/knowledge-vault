## 3. 横断的に見えてきた強い傾向

1. **議事録・文章作成は全主要ツールで圧倒的な削減効果**。Claude/Gemini/ChatGPT いずれも「2-8時間→5-10分」「半日→10分」「1時間→5分」レベルの短縮例が複数あり、削減率は概ね **80-95%**。
2. **OCR × LLM の組み合わせで紙・PDF業務がほぼ全自動化**。請求書（72h→4h、精度100%）・PDF→Excel・発注書 CSV化・紙就業規則検索など、有料OCRソフト不要化の事例が頻出。
3. **VBA/GAS/Python のコード自動生成で「ノンプログラマーの自動化」が当たり前に**。週1時間〜月数時間規模の小さな作業が瞬時化される例が多数。給与計算・シフト・補助金処理など定型業務の Gem 化・スキル化が定着段階に到達。
4. **「業務 → AIエージェント常駐」フェーズへの移行が始まっている**。Claude Code 夜間エージェント10個、Codex ローカル並行運用、Copilot 21エージェント量産、Gemini 予約アクションでの自動収集など、対話型から自律実行型への明確なシフト。
5. **大量データ処理の桁が変わった**。23万件TC矛盾チェック瞬時化、テレアポリスト2000件取得、5年分1000仕分け、社員経歴クレンジング 数ヶ月→2時間など、従来「人手では非現実的」だった作業が標準ユースケースに昇格。
6. **ツール役割分担の定着**：Claude（コード/長文/構造認識）、Gemini（Google連携/手書きOCR/長文）、ChatGPT（壁打ち/Projects/画像）、NotebookLM（参照型RAG）、Dify/n8n/GAS（フロー基盤）という棲み分けが、複数受講生で独立に同じ結論に収斂している。
