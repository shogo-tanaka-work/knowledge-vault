# MENTAプランページ サムネイル画像 生成プロンプト

作成日: 2026-05-05  
対象: ChatGPT Image（gpt-image-1 / 旧称 Imagine 2.0）  
用途: MENTAプラン `plan/20191` のメイン画像 / コース個別画像

---

## 設計コンセプト

### このサムネで伝えたい3つのメッセージ
1. **AI駆動開発の専門家**（Claude Code / Codex の2強を扱える）
2. **完成まで伴走する温度感**（独学挫折者を否定しない、親しみやすさ）
3. **数値の信頼感**（100名指導 / 評価5.0 / 4週間 / 5年経験）

### 競合との差別化ポイント
- 他の「Claude Code 教えます」系メンターは**ロゴ＋技術用語の冷たいバナー**が多い
- 省伍さんは「**独学挫折を否定しない伴走者**」がブランドコア
- → サムネは「**温かみのある手元 × プロのテック感**」のバランスが正解

### 推奨サイズ
- MENTAメイン画像: **横長（16:9 または 4:3 推奨）**
- ChatGPT Image でのアスペクト比指定: `aspect_ratio: 16:9` または `landscape`

## 🪄 バリエーション案（A/Bテスト用）

### バリエーション B: "二人で並んで進める" イメージ

```
シーン: 横並びの2つのモニターに、それぞれClaude CodeとCodexのコード画面。
画面の前にはマグカップが2つ、ノートが2冊。"先生と生徒が肩を並べて進む"
ことを連想させる構図。人物は写さない（手元すら写さない）。

意図: 「マンツーマン伴走」を物理的に表現。2台のモニター=2つのAIツールを
象徴し、"使い分け"のメッセージも込める。

その他は メインプロンプトと同じ雰囲気・色調・スタイル。
```

### バリエーション C: "完成までの道のり" イメージ

```
シーン: ホワイトボード or 紙のノートに、ラフな手書きで「企画 → 設計 → 実装 
→ デプロイ → 運用」のフローが描かれている。脇にラップトップが少し見えている。
紙のノートには小さく付箋（黄色）が貼られていて、メモ書きの跡がある。

意図: "完成まで伴走"の進行感をビジュアル化。ロードマップ感を伝える。

色調: 紙の白 × 鉛筆のグレー × 付箋の黄色 × ノートPCの濃いネイビー。
温かみのある柔らかい光。

避けるもの: 矢印が幼稚にならない（手書き感は保ちつつ整然と）。
```

### バリエーション D: "詰まりと解決" イメージ（少し攻めた案）

```
シーン: デスクの上に、開いたノートPCの画面に小さくエラーメッセージが見える。
その横で、Claude Code風のターミナルが解決策を提示している様子（コードの差分が
柔らかく光っている）。手前にはコーヒー、植物、ノート。

意図: "詰まったらすぐ聞ける"を象徴。独学挫折者の共感ポイントに直接触れる。

雰囲気・色調: メインプロンプトと同じ。"問題と解決が同居している" 静かな
ドラマチックさ。

避けるもの: エラー画面の文字が読めるほど大きく/鮮明に出さない（雰囲気で）。
```

## 📝 コース個別サムネ（小サイズ用・余裕があれば）

3コース別に、雰囲気を変えてもよい。

| コース | キービジュアル案 |
|--------|----------------|
| 🆓 無料相談 | 開いたノートとペン、湯気の立つマグカップ。"話しに来てね" の温度感。明るめ。 |
| 🧪 お試し体験 | キーボードに置かれた手の影（実際には手は写さない）、画面に少しコードが見える。"動かしてみる" の動的感。 |
| 🚀 本格実践 | 整理されたデスクとラップトップ、その奥に窓辺の風景。"長期で並走" の落ち着き。一番プロ寄り。 |

すべて同じ色調（ネイビー×アイボリー×ウォームブラウン）で**シリーズ感**を出すこと。

## 🚫 避けるべき表現（共通NG）

- ❌ AIロボット・サイボーグの擬人化（陳腐）
- ❌ 脳のホログラム・ニューラルネットワーク模式図（既視感）
- ❌ "GPT" "AI" などのテキストを画面に大きく入れる（権利・古さ）
- ❌ 過度なブルーグロー・ネオン（クールすぎて温度感ゼロ）
- ❌ ストックフォト感のあるビジネスマン握手・ガッツポーズ
- ❌ ターミナルの文字を読めるほど鮮明に（古いコードが映ると逆効果）
- ❌ 派手なグラデーション背景

## ✏️ 文字を入れる場合（後工程・Canva等で）

ChatGPT Image で文字を入れるとガタつく場合があるので、**画像生成は文字なし**で行い、後からCanva等でテキストオーバーレイするのが安定します。

オーバーレイ推奨文言:
```
メイン: Claude Code・Codexで、"動くアプリ"まで4週間伴走
サブ: 独学挫折OK / 100名指導 / 評価5.0
```

フォント推奨: Noto Sans JP（Bold）または游ゴシック（中太）。

## 🔁 試行のコツ

1. **まずメインプロンプト1案で4枚生成**して雰囲気を確認
2. 一番近いものを「`これに近い雰囲気で、〇〇を変更して`」と追加指示
3. ダメなら バリエーション B/C/D を試す
4. 文字オーバーレイ前提で、**中央〜右下に余白**ができる構図を選ぶ
5. 採用候補が決まったら、**横長16:9だけでなく正方形1:1も生成**しておく（SNS転用用）

```
MENTAのメンタープランページ用のメイン画像を作ってください。横長16:9。
木目調デスクに開かれたノートPC、Claude Code風のターミナル画面が柔らかく
光っている。脇に観葉植物と湯気の立つマグカップ。ネイビー×アイボリー×
```
